# 沝馬咖啡品牌官网

## 项目简介
为成都玉林社区“沝馬咖啡”搭建的品牌展示官网，主打温暖日系风格，展示品牌调性、菜单、咖啡豆产品及包场服务，帮助咖啡馆建立线上品牌形象并获取客户咨询。

## 项目信息
- **客户**：沝馬咖啡（成都玉林，3年独立咖啡馆）
- **类型**：网站 · 无后端
- **价格**：¥1,680
- **交付周期**：2-3天
- **技术栈**：HTML/CSS/JS（纯静态）

## 页面结构
- **首屏**：品牌slogan与氛围大图
- **关于我们**：品牌故事与核心亮点
- **菜单**：4款推荐饮品展示
- **咖啡豆**：挂耳包与咖啡豆电商展示
- **包场团建**：包场服务与手冲体验课介绍
- **联系地址**：门店信息、营业时间、咨询表单

## 功能特点
- 手机/电脑自适应，移动端优先设计
- 纯静态页面，零服务器成本，部署于 Cloudflare Pages
- 联系表单提交后即时反馈
- 配色温暖治愈，契合咖啡品牌调性

## 部署方式
本项目部署于 Cloudflare Pages，更新代码后自动同步上线。
